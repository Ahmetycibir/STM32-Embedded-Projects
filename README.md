# STM32-Embedded-Projects

**Collection of STM32-based embedded projects using sensors, actuators and modules such as RTC (DS3231), ultrasonic sensor, RFID, DHT11, servo motors, stepper motors, keypads, displays and more. A complete hands-on lab for microcontrollers and embedded systems.**

---

##  Project Overview

This repository contains a series of STM32F4-based projects designed to explore embedded systems, sensor interfacing, and actuator control.
Projects include beginner to advanced examples using:

* **RTC (DS3231)**
* **DHT11 temperature & humidity sensor**
* **Ultrasonic sensor (HC-SR04)**
* **RFID module (RC522)**
* **Servo motors (SG90)**
* **Stepper motors (28BYJ-48 + ULN2003)**
* **Keypads (4x4)**
* **LCD displays (1602 / I2C)**
* **LED matrix (8x8, MAX7219)**
* **Bluetooth modules (HC-05 / HC-06)**
* **Buzzer, push buttons, potentiometers, diodes, transistors**
